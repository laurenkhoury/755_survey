# 755_survey

Welcome!

## overview

This repo displays some of the work I did for a brief survey project as part of a graduate course in summer 2025. Our task was to administer a survey via Qualtrics and Prolific and use our developing data science skills to analyze the results. 

Inspired by my travels to Iceland, I became interested in studying work-life balance and how it might differ between cultures. I used both demographic data from Prolific and survey responses from Qualtrics to build an explanatory model for people's satisfaction to their work-life balance. 

## files

* `papers`

  - 2 reports from Iceland's longitudinal study on shortening the work week and its effects on participants
 
* `data`

  - wlb.csv: a de-indentified/reduced data file containing results from my work-life balance survey 
  - prolific data is not available here due to privacy concerns
 
* 755_workplace_survey.pdf: a printed version of the survey administered through the Qualtrics website

* survey.qmd: manuscript with code to report my results
* survey.pdf: rendered version of the qmd
